Controllers

class Controller

The HCCD-controller. Coordinates repeated CD-algorithm runs with state-space construction.

See also

Specialized versions for PCMCI-family time-series algorithms ControllerTimeseriesMCI and for LPCMCI ControllerTimeseriesLPCMCI are available.
